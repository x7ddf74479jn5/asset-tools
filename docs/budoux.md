# 🥋 BudouX

>BudouXは、美しく読みやすい改行を実現する、スタンドアロンで小型の、言語に依存しないフレーズセグメンターツールです。

```shell
Usage:
    yarn budoux [options]       テキストファイル内の文字列を整形したHTML文字列へ変換し、JSONとして出力
Options:
    --debug         dry-run
    --help, -h      ヘルプ
    -j              JSX形式に整形する
```
